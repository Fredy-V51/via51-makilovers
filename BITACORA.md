# 📓 Bitácora: Mi Transición a la Nube con Antigravity

Este documento resume los pasos realizados desde la configuración local hasta el despliegue en la nube.

## 1. Fase Local (El Origen)
* **Preparación:** Se creó el archivo `index.html` con un diseño moderno (gradientes y flexbox) en la PC local.
* **Uso de Git Gui:** Se utilizó para evitar el consumo excesivo de RAM. El proceso consistió en:
    * **Stage:** Seleccionar solo el archivo `index.html`.
    * **Commit:** Tomar la "foto" local del progreso.
    * **Push:** Enviar el archivo al repositorio `makilovers` en GitHub.

## 2. Fase de Documentación (README)
* **Identidad:** Se redactó un `README.md` que explica la visión del proyecto: unir la tecnología global con el legado y desarrollo del Perú.
* **Formato:** Se aplicó sintaxis Markdown para que sea legible y atractivo para humanos e inteligencias artificiales.

## 3. El Salto a la Nube (Codespaces)
* **Concepto:** Se activó una computadora virtual en los servidores de GitHub para dejar de depender del procesador de la PC física.
* **Acceso:** Se utiliza la tecla `.` (punto) para edición rápida o el botón verde **"<> Code" > Codespaces** para encendido completo.
* **Ventaja:** Permite ejecutar **Antigravity** sin interrupciones por falta de memoria.
